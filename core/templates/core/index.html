Provenance Web application

Implementing provenance data models using data from CosmoSim

The CosmoSim database stores a number of cosmological simulations. Read more about that project at CosmoSim.

This is a simple webapp for displaying provenance information of the simulation data, using the Django framework

The data were stored using different models, please pick one of them below:

  • W3C PROV-DM
  • IVOA ProvenanceDM
  • IVOA Simulation Data Model
