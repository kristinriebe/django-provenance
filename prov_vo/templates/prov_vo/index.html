Provenance Data for simulations in CosmoSim

The CosmoSim database stores a number of cosmological simulations. Read more about that project at CosmoSim.

This is a simple webapp for displaying provenance information of the simulation data, using the Django framework. It uses the data model currently discussed in the IVOA, which is based on the W3C ProvDM data model.

Pick one of the classes below to show the available records.

  • Activity
  • Entity
  • EntityDescriptions
